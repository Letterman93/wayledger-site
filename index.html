FAQ

What people will want to know.

Is Wayledger a mobile app?

Not at first. The first version will be a hosted web app that works across devices. Native apps can come later if users need them.

Will it connect to banks?

Bank connections are a later decision. The first version should prove the planning workflow before adding cost and complexity.

Can the public website be hosted for free?

Yes. The landing page can be hosted as a static site on GitHub Pages. Early-access collection can be handled with Google Forms, and contact mail can forward from hello@getwayledger.com.

When will subscriptions be available?

After tester feedback confirms the core workflows and the app has secure accounts, database setup, and support processes in place.
